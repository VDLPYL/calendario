Cuando se crea una nueva carpeta o un nuevo archivo, es obligatorio hacer:_
$ git add pages/
$ git commit -m "Se agrega un nuevo archivo"
no se puede hacer $ git commit -am "titulo", este comendo solo sirve para archivos modificados
-am = add y -m
