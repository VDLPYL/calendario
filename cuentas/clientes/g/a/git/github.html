¿Cómo subo un cambio a la nube en github?

1ro En mi pc, ingreso dentro de la carpeta de el proyecto o repositorio
2do Abro con la terminal GitBashHere
usuario@usuario-PC MINGW64 /d/VDLPYL/a/github/calendario (master)
3ro dentro de la terminal ingreso los siguientes comandos:
$ git init
$ ls
4to abro el proyecto con vscode
$ code .
$ git status
$ git log --oneline --all --graph --decorate
$ git add .
$ git commit -m "Genera el archivo de estilos"
$ git log --oneline --all --graph --decorate

Para subir es muy importante ingresar primero estos comandos
$ git remote
$ git remote add origin git@github.com:VDLPYL/calendario.git
$ git remote
origin {respuesta}
$ git branch
* master {respuesta}
$ git log
$ git status



$ git log --oneline --all --graph --decorate

$ git remote -v {importante para saber a que repositorio voy a subir los cambios}
$ git log --oneline --all --graph --decorate
$ git fetch origin {Es primero, para traer los cambios que hay en la nube}
$ git log --oneline --all --graph --decorate

Por tantos commits que he realizado en la nube, me aparecio :dos puntos en donde me dice que si quiero seguir mirando mas commits anteriores tengo que presionar la fecha hacia abajo o hacia arriba si quiero regresar
q {tengo que presionar la tecla "q" para salir y regresar al modo normal de la terminal}
$ git pull origin master {Es para traer los cambios de la nube, hasta la pc}

{m16:02/16:38 Entonces de esa manera ustedes pueden empujar código a la nube con $ git push origin y el nombre de la rama, pueden traerse información desde la nube con $ git fetch origin, y pueden jalarse información de la nube con $ git pull origin y el nombre de la rama }
